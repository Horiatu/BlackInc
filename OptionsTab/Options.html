Directions

Select your options here for text color and weight, then click the extension button to change the main dominant text on the page.


A shortcut menu-entry in the host-page popup-menu lets you select an element and remove it from the page. You may want this for removing advertisings or controls that block the main content, or to prepare the page for printing.

Locate the element you want to hide. With the mouse over, Right-Click. From the context menu, click 'BlackInk Little' menu entry:
